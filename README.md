# Student Management System (Java + MySQL)

## 📌 Project Description
This is a console-based Student Management System developed using Java and MySQL.  
It allows users to perform CRUD operations (Create, Read, Update, Delete) on student records.

## 🚀 Features
- Add Student
- View Students
- Update Student
- Delete Student
- Database connectivity using JDBC

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## 💡 Concepts Used
- Object-Oriented Programming (OOP)
- Database Management System (DBMS)
- Exception Handling

## 📂 Project Structure
src/com/sms
- Main.java
- StudentDAO.java
- DBConnection.java
